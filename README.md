
## Реализовано

- Использованы семантические теги: `header`, `nav`, `main`, `section`, `article`, `footer`
- На странице присутствует только один заголовок `h1`
- Все изображения подключены из папки `assets`
- HTML-код проходит проверку W3C Validator без критических ошибок
- Проект подготовлен к подключению CSS и шрифтов
